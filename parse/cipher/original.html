22-07-2024 Daily Cipher Morse Code Tips

To activate the Daily Cipher mode, select the “Daily Cipher” button on the “Exchange” tab (detailed instructions below)

Once the mode is activated, then you need to type the daily morse code by applying the patterns as follows.

  • Dash / long (— or – ) means a long tap for 3 seconds
  • Dot / short (. or • ) means a quick tap without holding your finger to the screen

Today, the Morse code is only four letters long and easy to type, so no special directions are required.

The Daily Cipher Code mode since July 21st is enabled directly by tapping the “Exchange Tab” at the bottom left corner of the screen and then selecting the “Daily Cipher” button at the middle top section of the screen.
